# SpendList

¡Bienvenido a SpendList, una app web para gestionar listas de compras y controlar gastos!

## Descripción

SpendList es una aplicación web moderna diseñada para crear y gestionar listas de compras con un enfoque especial en el control de gastos en tiempo real. Con una interfaz limpia y responsiva, permite a los usuarios organizarse mejor al momento de hacer compras, calculando automáticamente el total de gastos y proporcionando un desglose detallado por artículo. La aplicación cuenta con un modo demo que utiliza localStorage para persistencia temporal, así como integración con base de datos Turso (SQLite) para almacenamiento permanente. Además, incluye características modernas como modo oscuro, diseño responsivo y una experiencia de usuario optimizada para dispositivos móviles y desktop.

## Tecnologías a Utilizar

El proyecto se apoya en las siguientes tecnologías y herramientas modernas:

- **Astro y React**: Para construir una interfaz dinámica, modular y de alto rendimiento con renderizado híbrido.
- **TypeScript**: Para un desarrollo más robusto con tipado estático y mejor experiencia de desarrollo.
- **Tailwind CSS**: Para un diseño responsivo, moderno y altamente personalizable.
- **Radix UI**: Para componentes accesibles y de alta calidad como diálogos y selectores.
- **Turso (SQLite)**: Como base de datos serverless para el almacenamiento persistente de listas y artículos.
- **Lucide Icons**: Para iconografía moderna y consistente en toda la aplicación.
- **Git y GitHub**: Para el control de versiones y la colaboración en el desarrollo.
- **Vercel**: Para el despliegue y hosting de la aplicación con integración continua.

## Funcionalidades Clave

Entre las características más destacadas de SpendList se incluyen:

1. **Gestión Inteligente de Listas**
   - Creación, edición y eliminación de listas de compras personalizadas.
   - Organización por categorías y prioridades.

2. **Control de Gastos en Tiempo Real**
   - Cálculo automático del precio total con desglose por artículo.
   - Visualización de estadísticas de cantidad de artículos y totales.
   - Seguimiento del presupuesto planificado vs. gastado.

3. **Gestión de Artículos**
   - Agregar, editar y eliminar artículos de las listas.
   - Marcar artículos como comprados o pendientes.
   - Especificar precios, cantidades y notas adicionales.

4. **Experiencia de Usuario Moderna**
   - Modo oscuro/claro con alternancia intuitiva.
   - Diseño responsivo optimizado para móviles y desktop.
   - Interfaz limpia construida con componentes modernos.

5. **Almacenamiento Flexible**
   - Modo demo con localStorage para pruebas rápidas.
   - Integración con base de datos Turso para persistencia permanente.
   - Sincronización automática entre dispositivos.
